# juanmanuel.petrer.eu

Portfolio y blog personal de Juan Manuel Ruiz Muñoz — físico e ingeniero en IA especializado en visión computacional y deep learning.

## Stack

- [Hugo](https://gohugo.io/) — Generador de sitios estáticos
- [Tailwind CSS 4](https://tailwindcss.com/) — Framework de utilidades CSS
- JavaScript vanilla — Interacciones y animaciones
- [GitHub Pages](https://pages.github.com/) — Hosting

## Desarrollo local

```shell
npm install
hugo server
```

## Despliegue

```shell
hugo --gc --minify
git push origin master
```

El sitio se sirve desde la carpeta `docs/` vía GitHub Pages.
